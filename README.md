# Expense Tracker Web Application

This is a Full Stack Web Application built as a demo task.

The application allows users to manage expenses with full CRUD operations, view reports in a dashboard, and integrate with a third-party API for currency conversion.

---

## Tech Stack

Frontend:
- React.js
- CSS

Backend:
- FastAPI (Python)

Database:
- PostgreSQL / SQLite

Deployment:
- Frontend: Vercel
- Backend: Render

---

## Features

### 1. Full CRUD Operations

Users can:

- Create expenses
- View expenses
- Update expenses
- Delete expenses

CRUD operations work from:

- UI
- REST APIs

---

### 2. Dashboard / Reports

Dashboard shows:

- Total Expenses
- Expense Count
- Summary Metrics

Dashboard updates automatically when data changes.

---

### 3. Third Party API Integration

Currency conversion feature uses external API.

Users can:

- Select currency
- Convert expense amount

---
